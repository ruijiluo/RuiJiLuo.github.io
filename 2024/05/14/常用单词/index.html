常用单词

发表于2024-05-14|更新于2025-10-31
|阅读量:

在英语中，词的原型（通常为动词、形容词或名词）可以通过词缀（前缀或后缀）变化为其他词性。以下是一些常见的转换方式和例子：

1. 动词 (Verb) 转换为 名词 (Noun)

通过添加后缀，动词可以变成名词：

  • -tion / -sion / -ation / -ion
    • create → creation （创造 → 创造物）
    • decide → decision （决定 → 决定）
    • inform → information （通知 → 信息）
  • -ment
    • develop → development （发展 → 发展）
    • achieve → achievement （实现 → 成就）
  • -ing（动名词形式）
    • read → reading （阅读）
    • swim → swimming （游泳）
  • -er / -or（表示执行某一动作的人）
    • teach → teacher （教 → 教师）
    • act → actor （表演 → 演员）

2. 动词 (Verb) 转换为 形容词 (Adjective)

通过加后缀，动词可以变为形容词：

  • -able / -ible
    • read → readable （阅读 → 可读的）
    • believe → believable （相信 → 可信的）
    • access → accessible （访问 → 可访问的）
  • -ing（表进行中的动作）
    • excite → exciting （激动 → 令人兴奋的）
    • interest → interesting （感兴趣 → 有趣的）
  • -ed（表被动的状态或感觉）
    • bore → bored （使无聊 → 感到无聊的）
    • confuse → confused （使困惑 → 感到困惑的）

3. 名词 (Noun) 转换为 形容词 (Adjective)

名词可以通过后缀变成形容词：

  • -ful（表充满）
    • beauty → beautiful （美丽 → 美丽的）
    • joy → joyful （欢乐 → 充满欢乐的）
  • -less（表缺乏）
    • home → homeless （家 → 无家可归的）
    • hope → hopeless （希望 → 绝望的）
  • -ic / -ical
    • history → historic （历史 → 历史性的）
    • poet → poetic （诗人 → 诗意的）
  • -ous
    • danger → dangerous （危险 → 危险的）
    • fame → famous （名气 → 著名的）
  • -y
    • cloud → cloudy （云 → 多云的）
    • rain → rainy （雨 → 多雨的）

4. 形容词 (Adjective) 转换为 名词 (Noun)

形容词也可以通过加后缀变成名词：

  • -ness
    • happy → happiness （快乐 → 快乐的状态）
    • dark → darkness （黑暗 → 黑暗的状态）
  • -ity / -ty
    • active → activity （活跃 → 活动）
    • able → ability （能 → 能力）

5. 形容词 (Adjective) 转换为 副词 (Adverb)

形容词通过加后缀可以变成副词：

  • -ly
    • quick → quickly （快 → 快速地）
    • happy → happily （快乐 → 快乐地）

6. 名词 (Noun) 转换为 动词 (Verb)

有些名词通过加前缀或不加词缀就能变为动词：

  • -ize / -ise
    • modern → modernize （现代 → 现代化）
    • legal → legalize （合法 → 合法化）
  • -ify
    • beauty → beautify （美丽 → 美化）
    • simplify → simplify （简单 → 简化）
  • 零变形（名词直接用作动词）
    • email → to email （电子邮件 → 发邮件）
    • water → to water （水 → 浇水）

总结

在英语中，通过添加不同的前缀或后缀，一个词的原型（通常为名词、动词或形容词）可以转换为其他词性。这种灵活性使得英语能够根据上下文变化表达丰富的意思。
